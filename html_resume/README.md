# Personal Portfolio Page (HTML Assignment)

This project is a simple **personal portfolio webpage built using pure HTML and basic CSS**.  
It showcases personal information, skills, experience, projects, education, and contact details in a structured format.

The goal of this assignment is to practice **HTML fundamentals**, including semantic structure, lists, tables, and basic styling.

---

You can view the project here:

- Live Link: https://akghosh111.github.io/chai_code_cohort/html_resume/index.html
- GitHub Repo: https://github.com/akghosh111/chai_code_cohort/tree/main/html_resume

---

## Features

- Personal introduction section
- Skills list using HTML `<ul>`
- Experience section displayed in a **table**
- Projects section describing work and technologies used
- Education details in **table format**
- Contact information section
- Basic CSS styling for table borders

---

## Tech Stack

- **HTML5**
- **CSS3 (Basic styling)**

No frameworks or external libraries were used.
